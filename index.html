Create a complete GitHub project for a Registration Form website using HTML, CSS, and JavaScript. Project Requirements: - The project should work on GitHub Pages. - The main file should be named index.html. - The design should be clean, simple, and professional. - The layout should be responsive for desktop and mobile. The registration form should include the following fields: 1. Full Name 2. Gender (dropdown: Male, Female, Other) 3. Date of Birth 4. Phone Number 5. Email Address 6. Address 7. Qualification 8. Computer Skills 9. Work Experience 10. Submit Button Functionality Requirements: - Validate required fields. - When the user submits the form, prevent page refresh. - Display a success message like: "Registration Successful". - Clear the form after submission. Project Structure: registration-form/ ├── index.html ├── style.css └── script.js File Responsibilities: - index.html should contain the form structure. - style.css should contain modern and clean styling. - script.js should handle form submission and success message. Additional Requirements: - Use simple modern CSS. - Center the form on the page. - Add a card-style layout with shadow. - Use clear labels and spacing for each field.
